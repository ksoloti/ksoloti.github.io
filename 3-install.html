3. Install

Update: a standalone patcher for Ksoloti is currently being developed. If you decide to use it you don't need to do any of the below installation guides. You can download the beta for your OS here:

ksoloti_patcher-windows-1.0.12-3-113

ksoloti_patcher-mac-1.0.12-3-113

ksoloti_patcher-linux-1.0.12-3-113

Extract the package wherever you like and run the launcher file (Ksoloti.exe, Ksoloti.app, Ksoloti ...). Nothing will happen for a while until the Patcher has figured out who and where it is and has downloaded the libraries to your OSes user home folder.

The first time you connect your Ksoloti you will probably have to flash the standalone firmware using Rescue mode. Before connecting your Ksoloti, press and hold S1, then connect USB. The Ksoloti's LEDs will stay off. In the standalone patcher, select Board > Firmware > Flash (rescue) and wait for it to finish. If you're on Windows and the rescue Ksoloti is not recognized check out the info about Zadig in this guide.
